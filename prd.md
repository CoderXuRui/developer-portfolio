- 目的：作为个人技术沉淀的展示平台，建立个人品牌

- 目标受众：技术招聘方、合作伙伴、同行开发者

## 4. 技术选型

### 选定方案：Next.js + Tailwind CSS + Vercel 部署
- 静态站点生成（SSG），性能优秀
- Tailwind CSS 快速构建响应式 UI
- Vercel 一键部署

### 备选方案：
- Vite + React + 样式方案

## 5. 设计方向

- 辅助：#64748B（中性灰）

**字体：**

- 正文：Inter
